This PR replaces ~70 hardcoded prompt strings across 13 source files with a centralized, scope-aware template registry. Templates are stored in SQLite, resolved via a three-tier scope chain (`agent > repo > global`), and managed through REST API + MCP tools.

### Key Design Decisions

- **Scope precedence**: `agent > repo > global` (agents have final say on how they receive information)
- **Template shape**: Code-defined `header` (always-on essential event reference) + DB-customizable `body`

- **Three-state control**: `enabled | default_prompt_fallback | skip_event`
